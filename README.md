# Mix-Or-Match
## เกมจับคู่เพื่อการเรียนรู้
### พัฒนาเกมจับคู่เพื่อส่งเสริมทักษะด้านความจำและการคิดวิเคราะห์
### ออกแบบและพัฒนาเกมในรูปแบบเว็บแอปพลิเคชัน
### พัฒนาระบบตรวจสอบความถูกต้องและการนับคะแนนอัตโนมัติ
### ออกแบบส่วนติดต่อผู้ใช้ให้เข้าใจง่ายและเหมาะสมกับการใช้งาน
### สามารถนำไปประยุกต์ใช้เป็นสื่อการเรียนรู้หรือกิจกรรมเสริมทักษะได้
